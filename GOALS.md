# ChessLens 项目目标

## 核心目标

**主目标：孩子的棋力进步 + 社媒分享**

- 孩子（14岁）有自己的社媒账号，社媒分享是他的核心动力
- 项目帮助他记录、展示、复盘自己的棋局
- 目标：Chess.com 等级分从 1450 往上走

## 辅助目标

| 目标 | 说明 |
|------|------|
| 成长陪伴 | 2年内父子共同参与项目 |
| 孩子接手 | 16岁时他能维护、讲解这个项目 |
| 教学案例 | 供 AI 培训课演示用 |

## 内容形态

### 视频（社媒为主）

| 类型 | 定位 |
|------|------|
| 成长记录 | 主角，记录他棋力进步的过程 |
| 他的创作 | 棋局分析+故事+视觉风格，他主导 |

**主视频格式：** 长视频 3-5分钟，有叙事逻辑的复盘解说
**补充形式：** 复盘精华快剪（图文帖A是必须的）

### 复盘内容

- 直接可用于社媒发帖（不需要二次加工）
- 他自己主导内容创作，不只是被动看分析

## 核心定位

**差异化：复盘 + 社媒直出**

不与 chess.com 竞争通用 UX——那是商业产品，投入不是一个量级。

ChessLens 的价值在于：
- **复盘深度** — 比 chess.com 的分析更详细、更有教育意义、有人情味
- **社媒直出** — 复盘内容一键生成适合小红书/B站/抖音的图文/视频，不再是技术存档
- **成长叙事** — 有时间线的成长记录，不是零散对局

## 关键约束

- 孩子觉得 chess.com 比 ChessLens 更好用——不与其竞争通用 UX，聚焦差异化（复盘深度 + 社媒直出）
- 孩子目前没有社媒账号——需要一起建立的身份
- 视频/内容生产是他主导，不是你帮他做，是他来做
